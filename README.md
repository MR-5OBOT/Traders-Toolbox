## How to use

1. Clone the repository or download the zip file

```bash
git clone https://github.com/MR-5OBOT/probability-simulator.git
```

2. Make sure the app file is executable

```bash
chmod +x app
```

3. Install requirements

```bash
pip intstall -r requirements
```

4. Run the app file

```python
python3 app.py
```
